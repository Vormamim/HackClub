# Challenge 7: Score Tracker

**Focus:** variables, arithmetic

---

## Task Card
Create a program that:
- starts with a score of 0
- adds points for 3 actions
- prints the final score

Example:
- win round = +10
- bonus = +5
- mistake = -3

## Success Criteria
- uses a score variable
- changes the score more than once
- prints the final result

## Extension Task
- use `if` to give a medal message for high scores
