# Challenge 8: Mood Recommender

**Focus:** if / elif / else, practical output

---

## Task Card
Create a program that:
- asks the user how they feel: happy, tired, bored, stressed
- recommends an activity based on the answer

Example:
- happy → “Build something fun”
- tired → “Take a break”
- bored → “Try a mini game project”

## Success Criteria
- takes string input
- uses multiple decision branches
- outputs a recommendation

## Extension Task
- add 2 more moods
