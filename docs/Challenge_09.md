# Challenge 9: Playlist Picker

**Focus:** lists + conditions

---

## Task Card
Create a program that:
- has a list of music genres or video types
- asks the user what mood they are in
- recommends one genre or type of content

## Success Criteria
- uses at least one list
- uses decision logic
- outputs a suggestion based on input

## Extension Task
- recommend 2 options instead of 1
