# Challenge 4: Lucky Number Game

**Focus:** comparisons, simple logic

---

## Task Card
Create a program that:
- stores one lucky number in a variable
- asks the user to guess it
- tells them if they are correct

## Success Criteria
- uses a variable for the lucky number
- compares user input to the answer
- prints either success or failure

## Extension Task
- tell the user whether their guess was too high or too low
