# Challenge 2: Age Checker

**Focus:** if / else

---

## Task Card
Create a program that:
- asks for the user’s age
- tells them whether they are a child, teenager, or adult

## Success Criteria
- uses `if`, `elif`, `else`
- uses number comparison correctly
- gives different outputs for different ages

## Extension Task
- add a special message for 12-year-olds
