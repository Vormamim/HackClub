# Challenge 10: Digital Reward System

**Focus:** variables, lists, loop, decision logic

---

## Task Card
Create a program that:
- asks the user to complete 3 simple tasks
- gives points for each task completed
- awards a reward at the end

Example rewards:
- under 10 points = “Bronze”
- 10–20 = “Silver”
- above 20 = “Gold”

## Success Criteria
- uses variables
- uses a loop
- uses `if / elif / else`
- prints a final reward

## Extension Task
- store the task names in a list and loop through them
