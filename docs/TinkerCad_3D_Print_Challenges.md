# TinkerCad + 3D Printer Challenges

## Challenge 1: Custom Keychain

**Task:**  
Design a keychain with your name or initials.

**Success Criteria:**  
- Uses text tool in TinkerCad  
- Adds a hole for the keyring  
- Print is under 5cm

**Extension:**  
- Add a simple shape or icon

---

## Challenge 2: Cable Clip

**Task:**  
Create a small clip to organize charging cables.

**Success Criteria:**  
- Uses basic shapes (rectangle, cylinder)  
- Clip fits a standard USB cable  
- Print is under 4cm

**Extension:**  
- Make a double clip for two cables

---

## Challenge 3: Mini Phone Stand

**Task:**  
Design a tiny stand for a phone (portrait or landscape).

**Success Criteria:**  
- Uses angled shapes  
- Holds a phone upright  
- Print is under 6cm

**Extension:**  
- Add a slot for a charging cable

---

## Challenge 4: Dice

**Task:**  
Make a simple six-sided dice.

**Success Criteria:**  
- Uses cube shape  
- Adds dots or numbers  
- Print is under 2cm

**Extension:**  
- Make a pair of dice

---

## Challenge 5: Bag Tag

**Task:**  
Design a tag for a backpack or lunch bag.

**Success Criteria:**  
- Uses text and shapes  
- Adds a hole for string  
- Print is under 5cm

**Extension:**  
- Add a decorative border or icon
