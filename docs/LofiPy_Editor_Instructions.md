# LofiPy Editor: Basic Instructions

LofiPy is a browser-based Python editor with full syntax support and no backend server. Here’s how to use it:

## Getting Started
1. **Open the Editor:** Go to [vormamim.com/lofipy](https://vormamim.com/lofipy/).
2. **Activate:** Click anywhere on the page to activate the editor.
3. **Wait for Python to Load:** You’ll see a message when Python is ready.

## Main Features
- **Load .py:** Open a Python file from your computer.
- **Save .py:** Save your code as a .py file.
- **Clear:** Remove all code from the editor.
- **Run Code:** Execute your Python code. Output appears in the terminal below.
- **Font Size:** Use A-A+ to adjust text size.
- **Fullscreen:** Click ⛶ to expand the editor.

## Output Terminal
- Results and errors from your code will show here.
- Use "Clear" to reset the terminal.

## Python Environment
- Python version: 3.11.3
- Packages loaded: numpy, matplotlib
- Supports imports and standard Python syntax

## Tips
- No backend server: all code runs in your browser.
- Great for quick experiments and learning.
- Save your work often!
