# MicroPython + Microbit Challenges (DF Robot McQueen)

## Challenge 1: Simple Forward Drive

**Task:**  
Write a MicroPython program to make the McQueen drive forward for 2 seconds, then stop.

**Success Criteria:**  
- Uses motor control commands
- Drives straight and stops

**Extension:**  
- Make it reverse for 2 seconds after stopping

---

## Challenge 2: Obstacle Avoider

**Task:**  
Use the ultrasonic sensor to detect obstacles. If something is closer than 10 cm, stop and beep.

**Success Criteria:**  
- Reads distance from sensor
- Stops and beeps if too close

**Extension:**  
- Make it reverse and turn away from the obstacle

---

## Challenge 3: Line Follower

**Task:**  
Use the line sensors to follow a black line on white paper.

**Success Criteria:**  
- Reads line sensor values
- Adjusts direction to stay on the line

**Extension:**  
- Add a feature to stop at the end of the line

---

## Challenge 4: Light Show

**Task:**  
Use the micro:bit LEDs to display a pattern while the McQueen is moving.

**Success Criteria:**  
- Shows a pattern on the micro:bit display
- Pattern changes when the robot stops

**Extension:**  
- Make the pattern animate while moving

---

## Challenge 5: Remote Control

**Task:**  
Use the micro:bit’s buttons to control the McQueen:  
- Button A = forward  
- Button B = reverse  
- Both = stop

**Success Criteria:**  
- Responds to button presses
- Moves in correct direction

**Extension:**  
- Add speed control using gestures or more buttons
