# Challenge 3: School Day Helper

**Focus:** strings, decision making

---

## Task Card
Create a program that:
- asks what day of the week it is
- gives a different message depending on the day

Example:
- Monday → “Time to get organised”
- Friday → “You made it”
- Saturday/Sunday → “Enjoy the weekend”

## Success Criteria
- stores text input
- uses `if / elif / else`
- has at least 4 different possible outputs

## Extension Task
- include a message for a school sport day or club day
